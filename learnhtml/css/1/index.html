Fonts

Looking at a webpage you may have created, you might want to change the way it looks. You can do that with CSS.

body{
  font-family: sans-serif;
}

CSS stands for "cascading style sheets". The "cascading" part means that you can override attributes, which will be important soon.

What this bit of CSS does is change the font for everything in the <body> tag have a sans-serif font (like this one). The browser decides what font to display based on the user's selected default (in this case sans-serif) font. Your font selection can be more specific, like this:

body{
  font-family: "Roboto", sans-serif; }

In this case, the browser will attempt to display the webpage with the font "Roboto", and if that font is not found, it will fall back to the default sans-serif. It is very important that you add a fallback font. Without it, the browser will fall back to the default font, which is (usually) serif.
