The basics

Links:

Let's say you wanted to create a link to a certain webpage. You would do that like this:

<p>This <a href="https://example.com">here</a> is a link.</p>

Why is it an <a> tag and not a <link> tag? The a stands for anchor, and the href attribute indicates where it should point to. It should be a valid URL, either a complete one, like https://example.com, or a partial one, like /example2. The second will navigate to (siteName)/example2. Without the leading slash, it would navigate to (siteName)/(currentPage)/example2. The link tag is an unrelated tag, usually used to import stylesheets (later!).
