How to code HTML

The basics

The header:

You start off every HTML page with the following:

<html>
  <head>
    <meta charset="utf-8">
    <meta lang="en-US">
    <title>New Webpage</title>
  </head>
  More coming soon...

Here is what everything means:

  • <html>: The tag that tells the browser that what it is looking at is a webpage.
  • <head>: This marks the start of the header of the page. Nothing in here is displayed. Also, styling is put in this part of the page.
  • <meta>: Used to store metadata.
  • <title>: Stores the title that is displayed on the browser tab.

These tags are essential; your webpage will not work without them. All of this stuff is used to tell the browser what to do, not what to display. The only thing that will display from the header is the title.
